What is Wild?

Minecraft as nature intended.

🌍
Forever World
The world never resets. Everything that happens on Wild — every base built, every base raided — becomes a permanent part of its history.
⚔️
Unrestricted PvP
Full open-world PvP everywhere, all the time. No safe zones, no claims, no protection. Survive on your own terms.
🚫
No Economy
No coins, no shops, no admin shop. Resources have real value because the only way to get them is to gather, trade, or take them.
🏗️
No Claims
Nothing is protected by plugins. Build smart, build hidden, or build with allies. Your survival depends on you.
🔄
Always Updated
Wild always runs the latest version of Minecraft. New biomes, new blocks, new mobs — available the moment they release.
🛡️
Anti-Cheat Only
The only plugin running is anti-cheat. No gameplay modifications of any kind. As close to vanilla as a server can get.

One Rule

No hacking or cheating.

That's it. That's the only rule on Commons Wild. Everything else — raiding, stealing, PKing, building, alliances, betrayals — is fair game. The world is yours to shape however you choose.
